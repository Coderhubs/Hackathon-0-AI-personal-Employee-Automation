TROUBLESHOOTING
===============

"Login failed" for Gmail:
- Make sure you're using an App Password, not your main password
- Check if 2FA is enabled on your account
- Try logging in manually first to verify credentials

"Login failed" for LinkedIn:
- LinkedIn may require verification code
- Check your email/phone for the code
- Enter it in the browser window within 60 seconds
- LinkedIn may flag automated access as suspicious

Browser doesn't open:
- Check if antivirus is blocking Playwright
- Try running as administrator
- Make sure Chromium was installed: playwright install chromium

No emails/posts detected:
- Check logs in Gold_Tier/Logs/
- Gmail/LinkedIn may have changed their HTML structure
- The selectors may need updating
