================================================================================
                              HOW IT WORKS
================================================================================

THE WORKFLOW:
-------------
1. Watchers detect new items (Gmail, LinkedIn, WhatsApp)
2. Items saved to: Needs_Action/
3. Integration Coordinator processes them
4. Drafts created in: Pending_Approval/
5. YOU review and approve
6. Move approved files to: Approved/
7. Approval Handler executes actions
8. Completed items moved to: Done/
9. Dashboard updated automatically

THE FOLDERS:
-----------
Needs_Action/      - New items detected by watchers
Pending_Approval/  - Drafts awaiting YOUR review
Approved/          - YOU move approved files here
Done/              - Completed tasks archive
Plans/             - Execution plans
Logs/              - Activity logs

================================================================================
                          TROUBLESHOOTING
================================================================================

GMAIL NOT WORKING?
-----------------
- Check if 2FA enabled → Use App Password
  (Generate at: https://myaccount.google.com/apppasswords)
- Verify email in .env file
- Check browser window is logged in
- Restart Gmail watcher if needed

LINKEDIN NOT WORKING?
--------------------
- Verify login in browser window
- Check session not expired (re-login if needed)
- Increase check interval if rate-limited
- Restart LinkedIn watcher if needed

NO FILES IN PENDING_APPROVAL?
-----------------------------
- Check Needs_Action/ has files
- Wait 60 seconds for coordinator to process
- Check Logs/ for errors
- Verify watchers are running

BROWSER WINDOWS CLOSING?
------------------------
- Don't close browser windows manually
- Watchers need persistent browser sessions
- If closed, restart that specific watcher
- Check system sleep/hibernate settings
